# subtitles

## 目录说明

本目录用于存放影视内容字幕文件，按来源、语言与分辨率组织，便于检索与自动化处理。所有文件请保持统一命名与编码规范。

### 子目录
> 在你阅读本文时，下列子目录可能尚不存在

- [web/](./web/) — 流媒体或网络版本的字幕  
- [bluray/](./bluray/) — 蓝光版本的字幕  
- [dvd/](./dvd/) — DVD 版本的字幕



## Directory Overview

This directory contains subtitle files, organized by source, language, and resolution to facilitate discovery and automated processing. Please follow the repository's naming and encoding conventions for all files.

### Subdirectories
> The subdirectories listed below may not yet exist. 

- [web/](./web/) — Subtitles from streaming or digital releases  
- [bluray/](./bluray/) — Subtitles from Blu-ray releases  
- [dvd/](./dvd/) — Subtitles from DVD releases
