# en

## 目录说明
本目录存放数字发行的英文原始字幕，按分辨率与特性组织，便于检索与自动化处理。文件必须遵循[仓库命名规范](/docs/REPO_SUBTITLE_NAMING_CONVENTION.md)并使用 UTF-8 编码。


## Directory description
This directory contains original English subtitles for digital releases, organized by resolution and attributes for easy searching and automated processing. Files must follow [the repository naming conventions](/docs/REPO_SUBTITLE_NAMING_CONVENTION.md) and use UTF-8 encoding.
