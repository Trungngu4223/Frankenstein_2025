本目录存放来自网络、流媒体等数字发行的字幕，按语言进行分类组织，便于检索与自动化处理。应包含原始语言目录与（如有）对应的翻译目录，并非所有语言变体都会存在。请遵循仓库的命名与编码规范。

## Directory Overview

This directory contains subtitles from streaming and other digital releases, organized by language to support discovery and automated workflows. Each directory should include the source-language directory and, when available, a matching translation directory. Not all language variants are guaranteed to be present. Please follow the repository’s naming and encoding conventions.
